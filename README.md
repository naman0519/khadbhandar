#  Jai Shree Khad Bhandar - Fertilizer Website

A complete web-based solution for managing and showcasing fertilizer products online.  
This project is designed for local fertilizer shops to digitalize their business and accept orders online.

---

##  Features

- 🛒 View all fertilizer products (Urea, DAP, NPK, etc.)
- 📦 Product detail page
- 📝 Order placement system
- 👨‍💼 Admin panel (for managing products & orders)
- 📊 Backend API using Golang (Gin Framework)
- 🗄️ Database integration (PostgreSQL / MongoDB)
- 🌐 Responsive UI (works on mobile & desktop)

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Golang (Gin Framework)
- **Database:** PostgreSQL / MongoDB
- **Tools:** Git, GitHub
